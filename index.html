01

Melee Combat System

Created a responsive combat system in Unreal Engine 5 using Blueprints, Behavior Trees, and Blackboards to drive enemy interactions and gameplay flow. The AI features state-based behavior including patrolling, detecting the player, chasing, and engaging in combat with animation-driven attack logic. The system is built around clean decision-making structures and event-triggered responses, ensuring smooth transitions between combat states. This project demonstrates strong understanding of Unreal's AI gameplay pipeline, combat mechanics, and animation integration.

02

Toyota Supra MK4 Model

Created a highly detailed Toyota Supra MK4 3D model with a fully built interior and engine, optimized for real-time rendering. The vehicle is integrated into Unreal Engine 5 using the Chaos Vehicle system to enable physics-based driving and gameplay interaction. This project showcases skills in automotive modeling, texturing, and real-time integration, demonstrating the ability to bring complex assets from DCC tools into a functional game environment.

Endless Bike Game Prototype

Developed an endless bike run game in Unreal Engine 5, featuring a continuous highway system and physics-based vehicle handling powered by the Chaos Vehicle framework.
